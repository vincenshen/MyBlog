Blog WebSite

作者：沈洪斌
博客：http://www.cnblogs.com/vincenshen/

程序介绍：
	后端基于Django web框架
	前端基于bootstrap框架
	数据库为sqlite3
	
实现功能：
	用户管理：
		注册
		登录
		注销

	文章管理：
		新建
		查看
		修改
		删除
		分类
	
	搜索功能：
		对文章标题和内容关键字搜索
		
	Ajax实现：
		用户注册界面交互优化
		用户登录界面交互优化
		用户文章中心删除文章优化
	
	分页功能
		主页文章分页
		搜索结果分页
		
	
注意事项：
	Django版本为1.11.x
	
使用方法：
	启动服务端：pyhton manage.py  runserver
	首页地址： http://127.0.0.1
	默认用户： alex 123456a@
